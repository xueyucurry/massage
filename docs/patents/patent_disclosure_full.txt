《技术交底书完整版》初稿

一、发明名称
一种结合中医同身寸理论的人体背部膀胱经实时定位方法、系统及机器人设备

二、技术领域
本发明涉及中医智能诊疗装备、计算机视觉、人体姿态估计、时序点跟踪、深度三维重建及医疗机器人控制技术领域，尤其涉及一种将中医经络定位中的同身寸规则数字化，并与人体关键点检测、脊柱中线重建、遮挡跟踪恢复、深度三维映射和机器人轨迹生成相结合的膀胱经实时定位方法、系统及机器人设备。

三、背景技术
中医推拿治疗中，膀胱经是背部重要经络之一，点按、揉拨、分筋、顺筋等治疗手法通常围绕膀胱经及相关区域展开。传统中医经络定位多依据后正中线及其两侧的同身寸关系进行判断，即通过患者自身身体尺度确定经络与脊柱中线之间的横向距离。该方法体现了中医辨体施治和个体化定位的特点，但其主要依赖人工经验和触诊判断，难以直接用于机器人视觉识别和自动控制。

随着计算机视觉和机器人技术的发展，利用相机识别人背部结构并引导机械臂完成推拿操作成为可能。然而，现有技术在面向中医按摩机器人应用时仍存在如下不足：

1. 现有人体姿态检测方法主要输出通用人体关键点，难以直接对应中医经络位置。膀胱经并非直接可见的解剖结构，若仅依赖通用关键点或固定模板，难以获得符合中医定位逻辑的经络线。
2. 中医膀胱经定位强调同身寸，是一种随个体体型变化而变化的相对比例规则。现有视觉方法通常使用固定像素偏移、固定物理偏移或统一模板处理，不能准确反映中医个体化定位思想。
3. 在实际按摩场景中，患者可能处于侧卧、俯卧、半转体等姿态，且随着呼吸、肌肉松弛或轻微移动，背部区域会发生动态变化；此外，按摩头、机械臂末端或操作者手部还可能对局部区域产生遮挡。单纯依赖单帧检测的方法容易造成经络线抖动、漂移或丢失。
4. 现有很多视觉识别方案仅输出二维像素结果，即使能够在图像上显示近似经络位置，也无法直接为机器人生成具有空间坐标意义的执行轨迹。
5. 在机器人应用中，经络定位结果不仅要满足中医定位逻辑，还要满足轨迹连续、姿态稳定、可转换到机器人坐标系并可安全执行的要求。现有技术普遍缺少从中医定位规则到三维机器人路径之间的完整技术闭环。

因此，亟需一种能够将中医同身寸理论数字化、结构化，并结合人体关键点检测、时序跟踪、遮挡恢复、深度三维重建及机器人坐标变换，实现膀胱经实时稳定定位和机器人执行的技术方案。

四、发明目的
本发明的目的在于提供一种结合中医同身寸理论的人体背部膀胱经实时定位方法、系统及机器人设备，以解决现有技术中存在的经络定位难以个体化表达、遮挡和移动场景下稳定性不足、以及二维识别结果难以直接用于机器人执行的问题。

本发明进一步的目的在于：
1. 将中医经络定位中的同身寸规则转化为机器可计算的尺度参数；
2. 以人体背部脊柱中线为几何基准构建膀胱经位置模型；
3. 在局部遮挡和人体运动情况下持续输出稳定的膀胱经轨迹；
4. 将膀胱经检测结果直接转换为机器人坐标系下的三维轨迹；
5. 为中医按摩机器人提供点筋、分筋、顺筋等动作所需的路径基础。

五、技术方案
为实现上述目的，本发明采用如下技术方案：

一种结合中医同身寸理论的人体背部膀胱经实时定位方法，包括如下步骤：

1. 采集人体背部的彩色图像和深度图像，并将深度图像对齐到彩色图像坐标系。
2. 对所述彩色图像执行人体姿态估计，提取肩部关键点和髋部关键点。优选地，对多个图像朝向分别执行姿态估计，以提高侧卧、倒置和复杂姿态下的检出稳定性。
3. 根据所述肩部关键点和髋部关键点构造脊柱中线。优选地，所述脊柱中线并非简单由肩中点与髋中点直接连线获得，而是结合肩部和髋部整体几何关系、主方向估计及异常情况判别生成。
4. 将中医同身寸规则转化为个体化尺度参数 L。所述个体化尺度参数 L 可由人工输入、视觉测量、历史档案读取或其组合方式获得。
5. 以所述脊柱中线为基准，沿人体横向方向在脊柱中线两侧按比例偏移生成膀胱经轨迹线。优选地，生成位于第一同身寸倍数位置的内侧膀胱经线，以及位于第二同身寸倍数位置的外侧膀胱经线。
6. 沿所述脊柱中线均匀设置多个离散采样点，并对所述离散采样点执行时序跟踪。优选地，所述跟踪对象为脊柱中线上的离散点集，而非整个人体区域。
7. 当人体发生移动、姿态变化或局部遮挡时，根据当前仍可见的采样点、历史脊柱中线及异常点剔除机制，对当前帧脊柱中线进行重建，并依据重建后的脊柱中线和个体化尺度参数重新生成膀胱经轨迹线。
8. 对膀胱经轨迹线执行稳定化处理，所述稳定化处理包括至少以下一种或多种：单帧跳变限制、短时保持、重获取加权、稳定帧判定和可执行状态判定。
9. 对膀胱经轨迹线上的多个像素点执行深度采样，在其邻域内提取多个深度值并进行滤波处理，基于相机内参将所述像素点反投影为相机坐标系下的三维点。
10. 通过相机坐标系至机器人基坐标系的标定矩阵，将所述三维点转换为机器人基坐标系下的三维轨迹。
11. 将所述三维轨迹输出至机器人控制模块，用于生成中医按摩机器人的点筋、分筋、顺筋、点按或滑移操作路径。

进一步地，在一个优选实施方式中，个体化尺度参数 L 为一指同身寸数值，膀胱经轨迹线依据“脊柱中线 + 同身寸倍数偏移”的方式生成。

进一步地，在一个优选实施方式中，外侧膀胱经线根据内侧膀胱经线相对于脊柱中线的比例扩展得到。

进一步地，在一个优选实施方式中，深度采样采用目标像素邻域中值深度，以减小单像素噪声和空洞深度带来的轨迹抖动。

进一步地，在一个优选实施方式中，相机坐标系至机器人基坐标系的变换通过带标记物的空间点对标定获得，所述变换为四维齐次刚体变换矩阵。

本发明还提供一种结合中医同身寸理论的人体背部膀胱经实时定位系统，包括：
图像采集模块，用于采集彩色图像和深度图像；
姿态估计模块，用于提取肩部关键点和髋部关键点；
同身寸参数生成模块，用于生成个体化尺度参数；
脊柱中线构造模块，用于生成或重建脊柱中线；
经络轨迹生成模块，用于依据脊柱中线和个体化尺度参数生成膀胱经轨迹线；
时序跟踪模块，用于对脊柱中线离散采样点进行连续跟踪；
轨迹稳定模块，用于对经络轨迹进行状态保持和稳定判定；
三维映射模块，用于将二维经络轨迹转换为三维轨迹；
坐标变换模块，用于将相机坐标系下的三维轨迹转换为机器人基坐标系下的轨迹；
机器人接口模块，用于输出机器人可执行路径。

本发明还提供一种中医按摩机器人设备，所述设备包括上述系统、机器人本体、末端按摩执行器及控制器，所述控制器用于根据所述膀胱经三维轨迹生成按摩动作控制指令。

本发明还提供一种存储介质，其上存储有计算机程序，所述程序在处理器上执行时，实现上述方法步骤。

六、有益效果
与现有技术相比，本发明具有以下有益效果：

1. 本发明将中医同身寸理论中的经验性定位规则参数化为机器可执行的尺度参数，使中医经络定位规则可以进入视觉识别和机器人控制链路。
2. 本发明以脊柱中线为结构基准对膀胱经进行建模，避免直接将通用人体关键点等同于经络位置，提高了中医语义一致性。
3. 本发明采用脊柱离散点跟踪而非整人体区域跟踪，在遮挡和人体运动条件下仍可利用可见点重建脊柱中线并恢复膀胱经轨迹，具有较强的抗遮挡能力和连续性。
4. 本发明通过稳定化机制对经络线进行跳变抑制和短时保持，能够显著减少经络轨迹闪跳、漂移和瞬时丢失，满足实时机器人控制的稳定性要求。
5. 本发明结合深度反投影和手眼标定变换，实现从二维经络线到机器人三维执行轨迹的直接转换，形成完整闭环，便于中医按摩机器人直接执行。
6. 本发明适用于中医按摩机器人、推拿辅助设备、康复机器人及中医教学演示等多种场景，具有良好的工程应用价值。

八、具体实施方式
以下结合优选实施例，对本发明技术方案作进一步说明。

1. 系统构成
本实施例中的系统包括 RGB-D 图像采集装置、视觉处理单元、三维重建与坐标转换单元、机器人控制单元和人机交互单元。RGB-D 图像采集装置用于同步采集人体背部彩色图像和深度图像；视觉处理单元用于执行人体关键点检测、脊柱中线构建、时序跟踪和经络轨迹生成；三维重建与坐标转换单元用于将图像中的膀胱经轨迹转换为机器人基坐标系下的三维轨迹；机器人控制单元用于根据所述轨迹生成点筋、分筋、顺筋或滑移按摩路径；人机交互单元用于输入同身寸尺度参数、治疗参数和工作模式。

2. 同身寸尺度参数的数字化
中医膀胱经定位强调同身寸，即以患者自身身体尺度作为定位依据。为了使该中医规则能够被机器执行，本实施例将同身寸规则转化为个体化尺度参数 L。

在一个实施方式中，个体化尺度参数 L 由操作者输入，输入值可为患者一指宽对应的毫米数值。

在另一实施方式中，个体化尺度参数 L 由视觉系统根据人体局部宽度自动估计得到。例如，可利用肩部宽度、髋部宽度、背部局部宽度或这些尺度的组合，通过比例关系换算得到一指同身寸的等效长度。

进一步地，为使同身寸参数在图像平面和物理空间中均可使用，本实施例中分别引入像素尺度下的同身寸偏移量和三维物理空间下的同身寸偏移量。视觉检测阶段可先在二维图像中使用像素尺度偏移量生成经络候选线；在获得深度信息后，再将其映射到物理空间中，以便形成三维经络轨迹。

3. 背部关键点检测与脊柱中线构建
本实施例首先对人体背部彩色图像执行人体关键点检测，优选提取左肩、右肩、左髋和右髋关键点。

考虑到中医按摩场景中患者可能处于侧卧、翻转或局部倾斜状态，本实施例优选对原始图像及多个旋转朝向图像分别执行姿态检测，并选取关键点置信度最高的检测结果作为当前帧姿态结果。

在获得肩部和髋部关键点后，本实施例不直接采用肩中点与髋中点之间的连线作为脊柱中线，而是：
1. 计算左右肩关键点的中点与左右髋关键点的中点；
2. 计算躯干的纵向主方向；
3. 结合肩部宽度、髋部宽度和躯干整体主轴对脊柱方向进行修正；
4. 对髋部关键点退化、躯干长度异常缩短或肩髋关系明显失真的情形进行异常判别。

4. 膀胱经生成
在获得脊柱中线后，本实施例以所述脊柱中线作为中医后正中线的近似几何表达。然后依据个体化尺度参数 L 和人体横向方向，在脊柱中线两侧生成膀胱经轨迹线。

在一个优选实施方式中，在脊柱中线两侧分别按第一比例系数 k1 与尺度参数 L 的乘积进行偏移，生成第一对膀胱经线，作为内侧膀胱经线；再按第二比例系数 k2 与尺度参数 L 的乘积进行偏移，生成第二对膀胱经线，作为外侧膀胱经线，其中 k2 大于 k1。

在一个优选实施方式中，外侧膀胱经线不是重新独立估计，而是以内侧膀胱经线为基础，相对于脊柱中线做比例扩展得到。

5. 脊柱离散采样点的时序跟踪
本实施例并不直接对整个人体区域或整条经络线进行跟踪，而是仅沿脊柱中线布设多个离散采样点，并对这些离散采样点执行时序跟踪。

具体地，沿当前脊柱中线从上端至下端均匀采样，得到多个离散采样点。随后，将这些离散采样点输入时序点跟踪模型，以获得连续多帧中的采样点运动轨迹和可见性信息。

在一个优选实施例中，脊柱中线上的离散采样点数可设置为 20 至 60 个，优选约 40 个。

6. 遮挡和运动过程中的脊柱重建
在时序跟踪过程中，本实施例对可见脊柱采样点执行如下重建处理：
1. 获取当前帧脊柱采样点的坐标及可见性信息；
2. 剔除相对于历史脊柱轨迹偏差过大的异常点；
3. 对剩余可见点执行主方向估计，以恢复当前帧脊柱主轴方向；
4. 按脊柱纵向参数重建当前帧脊柱点列；
5. 将当前帧重建结果与上一稳定帧结果进行平滑融合。

当遮挡比例较低时，可优先采用当前可见点重建的脊柱结果；当遮挡比例较高但尚未完全失效时，可保留上一稳定帧结果并进入保持状态；当连续多帧均无法满足重建条件时，再重置脊柱跟踪状态，重新等待检测初始化。

7. 稳定化处理
为了满足机器人控制需要，本实施例还对膀胱经轨迹线执行稳定化处理，包括：
1. 单帧最大跳变限制；
2. 时序平滑；
3. 短时缺失时维持上一稳定经络线；
4. 重新获取后对新旧结果进行加权融合；
5. 连续若干帧稳定后才判定为可执行状态。

8. 二维经络线到三维经络轨迹的转换
对经络线上的多个像素点进行均匀采样。对每个采样像素点，在深度图的对应邻域内提取多个深度值，并通过中值滤波得到稳定深度值。之后，基于相机内参执行像素反投影，得到相机坐标系下的三维空间点。对多个经络采样点重复该过程，得到相机坐标系下的膀胱经三维点列。

9. 相机坐标系到机器人基坐标系的变换
在一个优选实施例中，采用带标记物的空间点对标定方式，具体包括：
1. 在机器人末端或标定载体上设置已知标记物；
2. 通过相机识别标记物在相机坐标系中的空间位置；
3. 同时读取机器人末端在机器人基坐标系中的位置；
4. 采集多组对应点对；
5. 根据所述多组点对求解相机坐标系至机器人基坐标系的刚体变换矩阵。

为提高运行时一致性，优选使用与实际检测阶段一致的三维获取方式，即采用深度反投影得到相机坐标。

10. 按摩机器人中的路径生成
在一个面向中医按摩机器人的实施例中，机器人基坐标系下的膀胱经轨迹点列可进一步用于生成按摩动作路径，优选包括：
1. 根据经络点列计算轨迹切向方向；
2. 根据局部体表法向估计末端执行器接触方向；
3. 根据经络切向和法向生成分筋方向；
4. 由经络点、接触方向和分筋方向共同确定机器人末端姿态。

在一个实施方式中，可沿外侧膀胱经生成多个按摩点，依次执行点筋、分筋和顺筋动作。

11. 实施效果
采用本发明技术方案后，可以实现：
1. 基于同身寸参数的个体化膀胱经定位；
2. 在侧卧、局部遮挡和人体微动条件下持续输出稳定经络轨迹；
3. 将检测结果直接转换为机器人可执行的三维路径；
4. 支撑中医按摩机器人完成点筋、分筋和顺筋动作。

九、创新点提炼
1. 将中医同身寸理论参数化为机器人可执行的个体化尺度参数。
2. 将膀胱经定位建模为“脊柱中线重建 + 同身寸比例偏移”问题。
3. 采用脊柱离散采样点而非整人体区域执行时序跟踪。
4. 在遮挡或移动条件下，根据可见采样点重建脊柱并恢复膀胱经轨迹。
5. 将经络实时定位结果直接转换为机器人基坐标系下的三维执行轨迹。

十、可保护的核心点建议
1. 同身寸参数化的经络定位方法。
2. 基于脊柱中线的双层膀胱经生成方法。
3. 基于脊柱离散采样点的遮挡恢复方法。
4. 二维经络轨迹到机器人三维路径的统一转换方法。
5. 面向中医按摩机器人的经络定位与执行一体化方案。
