《说明书摘要 + 背景技术 + 发明内容》正式稿

一、说明书摘要
本发明公开了一种结合中医同身寸理论的人体背部膀胱经实时定位方法、系统及设备，属于中医智能机器人、计算机视觉与医疗装备技术领域。该方法通过 RGB-D 相机采集人体背部彩色图像和深度图像，对人体执行姿态估计以提取肩部关键点和髋部关键点，并基于所述关键点构造脊柱中线；进一步将中医经络定位中的同身寸规则转化为个体化尺度参数，以所述脊柱中线为基准，在其两侧按比例生成膀胱经轨迹线；沿脊柱中线设置多个离散采样点并执行时序跟踪，在人体移动、姿态变化或局部遮挡条件下，根据可见采样点重建脊柱中线并恢复膀胱经轨迹线；再结合深度反投影和相机至机器人基坐标系的标定变换，将膀胱经轨迹线转换为机器人可执行的三维路径。该发明兼顾中医理论一致性、实时性、抗遮挡能力和机器人可执行性，适用于中医按摩机器人、推拿辅助机器人、经络定位设备及中医教学演示系统。

二、背景技术
中医推拿治疗中，膀胱经是背部重要经络之一，点按、揉拨、分筋、顺筋等治疗手法通常围绕膀胱经及其相关区域展开。传统中医经络定位多依据后正中线及其两侧的同身寸关系进行判断，即通过患者自身身体尺度确定经络与脊柱中线之间的横向距离。该方法体现了中医辨体施治和个体化定位的特点，但其主要依赖人工经验、触诊判断和临床训练，难以直接用于机器人视觉识别和自动控制。

随着机器人技术和计算机视觉技术的发展，利用相机识别人背部结构并引导机械臂完成推拿操作成为可能。然而，现有技术在面向中医按摩机器人应用时仍存在如下不足。

1. 现有人体姿态检测方法主要输出通用人体关键点，难以直接对应中医经络位置。膀胱经并非直接可见的解剖结构，若仅依赖通用关键点或固定模板，难以获得符合中医定位逻辑的经络线。
2. 中医膀胱经定位强调同身寸，是一种随个体体型变化而变化的相对比例规则。现有视觉方法通常使用固定像素偏移、固定物理偏移或统一模板处理，不能准确反映中医个体化定位思想。
3. 在实际按摩场景中，患者可能处于侧卧、俯卧、半转体等姿态，且随着呼吸、肌肉松弛或轻微移动，背部区域会发生动态变化；此外，按摩头、机械臂末端或操作者手部还可能对局部区域产生遮挡。单纯依赖单帧检测的方法容易造成经络线抖动、漂移或丢失。
4. 现有很多视觉识别方案仅输出二维像素结果，即使能够在图像上显示近似经络位置，也无法直接为机器人生成具有空间坐标意义的执行轨迹。
5. 在机器人应用中，经络定位结果不仅要“看起来正确”，还要满足轨迹连续、姿态稳定、可转换到机器人坐标系并可安全执行的要求。现有技术普遍缺少从中医定位规则到三维机器人路径之间的完整技术闭环。

因此，亟需一种能够将中医同身寸理论数字化、结构化，并结合人体关键点检测、遮挡情况下的时序跟踪恢复、深度三维重建与机器人坐标变换，实现膀胱经实时稳定定位的技术方案。

三、发明内容

（一）发明目的
本发明的目的在于提供一种结合中医同身寸理论的人体背部膀胱经实时定位方法、系统及设备，以解决现有技术中存在的经络定位难以个体化表达、遮挡和移动场景下稳定性不足、以及二维识别结果难以直接用于机器人执行的问题。

本发明进一步的目的在于：
1. 将中医经络定位中的同身寸规则转化为机器可计算的尺度参数；
2. 以人体背部脊柱中线为几何基准构建膀胱经位置模型；
3. 在局部遮挡和人体运动情况下持续输出稳定的膀胱经轨迹；
4. 将膀胱经检测结果直接转换为机器人坐标系下的三维轨迹，用于中医按摩机器人执行。

（二）技术方案
为实现上述目的，本发明采用如下技术方案。

本发明提供一种结合中医同身寸理论的人体背部膀胱经实时定位方法，包括如下步骤：

1. 采集人体背部的彩色图像和深度图像，并将深度图像对齐到彩色图像坐标系。
2. 对所述彩色图像执行人体姿态估计，提取肩部关键点和髋部关键点；优选地，可对多个图像朝向分别进行姿态估计，以提高侧卧、倒置及复杂姿态下的检出稳定性。
3. 根据所述肩部关键点和髋部关键点构造脊柱中线。优选地，所述脊柱中线并非简单由肩中点与髋中点直接连线获得，而是结合肩部和髋部整体几何关系、主方向估计及异常情况判别生成，以提升脊柱中线构造的鲁棒性。
4. 将中医同身寸规则转化为个体化尺度参数 L。所述个体化尺度参数 L 可由人工输入、视觉测量、历史档案读取或其组合方式获得。
5. 基于所述脊柱中线和个体化尺度参数 L，在脊柱中线两侧沿人体横向方向按比例偏移生成膀胱经轨迹线。优选地，可生成位于第一同身寸倍数位置的内侧膀胱经线，以及位于第二同身寸倍数位置的外侧膀胱经线。
6. 沿所述脊柱中线均匀设置多个离散采样点，并对所述离散采样点执行时序跟踪。所述跟踪对象优选为脊柱中线上的离散点集，而非整个人体框区域。
7. 当人体发生移动、姿态变化或局部遮挡时，根据当前仍可见的采样点、历史脊柱中线及异常点剔除机制，对当前帧脊柱中线进行重建，并依据重建后的脊柱中线和个体化尺度参数重新生成膀胱经轨迹线。
8. 对膀胱经轨迹线执行稳定化处理，所述稳定化处理包括至少以下一种或多种：单帧跳变限制、短时保持、重获取加权、稳定帧判定和可执行状态判定。
9. 对膀胱经轨迹线上的多个像素点执行深度采样，在其邻域内提取多个深度值并进行滤波处理，基于相机内参将所述像素点反投影为相机坐标系下的三维点。
10. 通过相机坐标系至机器人基坐标系的标定矩阵，将所述三维点转换为机器人基坐标系下的三维轨迹。
11. 将所述三维轨迹输出至机器人控制模块，用于生成中医按摩机器人的点筋、分筋、顺筋、点按或滑移操作路径。

进一步地，在一个优选实施方式中，所述个体化尺度参数 L 为一指同身寸数值，膀胱经轨迹线依据“脊柱中线 + 同身寸倍数偏移”的方式生成。
进一步地，在一个优选实施方式中，所述外侧膀胱经线根据内侧膀胱经线相对于脊柱中线的比例扩展得到。
进一步地，在一个优选实施方式中，所述深度采样采用目标像素邻域中值深度，以减小单像素噪声和空洞深度带来的轨迹抖动。
进一步地，在一个优选实施方式中，所述相机坐标系至机器人基坐标系的变换通过带标记物的空间点对标定获得，所述变换为四维齐次刚体变换矩阵。

本发明还提供一种结合中医同身寸理论的人体背部膀胱经实时定位系统，包括：
图像采集模块，用于采集彩色图像和深度图像；
姿态估计模块，用于提取肩部关键点和髋部关键点；
同身寸参数生成模块，用于生成个体化尺度参数；
脊柱中线构造模块，用于生成或重建脊柱中线；
经络轨迹生成模块，用于依据脊柱中线和个体化尺度参数生成膀胱经轨迹线；
时序跟踪模块，用于对脊柱中线离散采样点进行连续跟踪；
轨迹稳定模块，用于对经络轨迹进行状态保持和稳定判定；
三维映射模块，用于将二维经络轨迹转换为三维轨迹；
坐标变换模块，用于将相机坐标系下的三维轨迹转换为机器人基坐标系下的轨迹；
机器人接口模块，用于输出机器人可执行路径。

本发明还提供一种中医按摩机器人设备，所述设备包括上述系统、机器人本体、末端按摩执行器及控制器，所述控制器用于根据所述膀胱经三维轨迹生成按摩动作控制指令。

本发明还提供一种存储介质，其上存储有计算机程序，所述程序在处理器上执行时，实现上述方法步骤。

（三）有益效果
与现有技术相比，本发明具有以下有益效果：

1. 本发明将中医同身寸理论中的经验性定位规则参数化为机器可执行的尺度参数，使中医经络定位规则可以进入视觉识别和机器人控制链路。
2. 本发明以脊柱中线为结构基准对膀胱经进行建模，避免直接将通用人体关键点等同于经络位置，提高了中医语义一致性。
3. 本发明采用脊柱离散点跟踪而非整人体区域跟踪，在遮挡和人体运动条件下仍可利用可见点重建脊柱中线并恢复膀胱经轨迹，具有较强的抗遮挡能力和连续性。
4. 本发明通过稳定化机制对经络线进行跳变抑制和短时保持，能够显著减少经络轨迹闪跳、漂移和瞬时丢失，满足实时机器人控制的稳定性要求。
5. 本发明结合深度反投影和手眼标定变换，实现从二维经络线到机器人三维执行轨迹的直接转换，形成完整闭环，便于中医按摩机器人直接执行。
6. 本发明适用于中医按摩机器人、推拿辅助设备、康复机器人及中医教学演示等多种场景，具有良好的工程应用价值。
