# lasttime.py 使用说明

## 功能概述

lasttime.py 是一个膀胱经按摩动作演示程序，展示基于 CoTracker 的四条膀胱经检测成果，并让机械臂在左外侧膀胱经上执行模拟按摩动作。

## 按摩动作

程序会在左外侧膀胱经上取10个点，依次执行以下动作：

### 1. 点筋动作
- 末端向更接近背部方向移动（Z轴负方向）
- 默认深度：20mm
- 每个点停留0.5秒

### 2. 分筋动作
- 沿身体侧向（垂直于脊柱）左右移动
- 默认偏移：20mm
- 动作序列：中心 → 左侧 → 右侧 → 中心

### 3. 顺筋动作
- 从第一个点到最后一个点线性移动
- 保持悬空高度
- 展示完整轨迹跟随

## 使用方法

### 基本使用

```bash
./run_lasttime.sh
```

### 自定义参数

```bash
# 设置悬空高度为30mm
HOVER_HEIGHT_MM=30 ./run_lasttime.sh

# 设置点筋深度为15mm
DIAN_JIN_DEPTH_MM=15 ./run_lasttime.sh

# 设置分筋偏移为25mm
FEN_JIN_LATERAL_MM=25 ./run_lasttime.sh

# 组合设置
HOVER_HEIGHT_MM=30 DIAN_JIN_DEPTH_MM=15 FEN_JIN_LATERAL_MM=25 ./run_lasttime.sh
```

### 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| ROBOT_IP | 机械臂IP地址 | 192.168.58.2 |
| HOVER_HEIGHT_MM | 悬空高度（mm） | 25.0 |
| DIAN_JIN_DEPTH_MM | 点筋深度（mm） | 20.0 |
| FEN_JIN_LATERAL_MM | 分筋横向偏移（mm） | 20.0 |

## 运行流程

1. **初始化阶段**（约5秒）
   - 启动RealSense相机
   - 加载YOLO姿态检测模型
   - 加载相机到机械臂的标定矩阵

2. **检测稳定阶段**（约3-10秒）
   - 实时检测四条膀胱经
   - 等待连续8帧稳定
   - 画面显示检测结果和稳定进度

3. **轨迹采集阶段**（约1秒）
   - 在左外侧膀胱经上采样10个点
   - 转换为机械臂基座坐标
   - 计算侧向向量用于分筋动作

4. **机械臂执行阶段**（约2-3分钟）
   - 连接机械臂
   - 移动到安全高度
   - 依次执行10个点的点筋+分筋动作
   - 回到起点执行顺筋动作
   - 返回安全位置

## 安全注意事项

1. **悬空模拟**：所有动作保持悬空，不接触人体
2. **安全高度**：转场时机械臂会移动到300mm安全高度
3. **速度控制**：
   - 转场速度：30%
   - 按摩动作速度：10%
4. **紧急停止**：按下机械臂急停按钮可立即停止

## 故障排除

### 检测超时
- **原因**：人体姿态不清晰或光线不足
- **解决**：调整人体位置，确保肩部和髋部清晰可见

### 深度数据不足
- **原因**：深度相机无法获取有效深度
- **解决**：调整相机角度，避免反光表面

### 标定矩阵缺失
- **原因**：camera_to_robot.json文件不存在
- **解决**：运行标定程序生成标定矩阵

### 机械臂连接失败
- **原因**：IP地址错误或网络不通
- **解决**：检查ROBOT_IP环境变量和网络连接

### 运动失败（err=112）
- **原因**：目标位姿超出机械臂工作空间
- **解决**：调整相机位置或减小动作幅度

## 技术细节

### 坐标转换流程

```
像素坐标 (u, v)
    ↓ [深度采样]
深度值 (米)
    ↓ [反投影]
相机3D坐标 [x, y, z] (米)
    ↓ [4x4变换矩阵]
机械臂基座坐标 [x, y, z] (米)
    ↓ [×1000]
机械臂基座坐标 [x, y, z] (毫米)
```

### 膀胱经定义

- **内侧膀胱经**：距离脊柱1.5倍手指宽（约45mm）
- **外侧膀胱经**：距离脊柱3倍手指宽（约90mm）
- **左外侧膀胱经**：视觉上的最上边一条，用于本演示

|------|----------|----------|----------|

| 安全测试 | 40-50mm | 15-20mm | 15-20mm |
| 精细调试 | 20-25mm | 15-20mm | 15-20mm |
