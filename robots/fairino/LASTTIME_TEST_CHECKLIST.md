## 测试步骤

### 1. 语法检查
```bash
python3 -m py_compile lasttime.py
```
- [ ] 无语法错误

### 2. 视觉检测测试（不连接机械臂）
修改代码临时跳过机械臂初始化，只测试视觉部分：
- [ ] 相机正常启动
- [ ] YOLO模型加载成功
- [ ] 标定矩阵加载成功
- [ ] 能检测到四条膀胱经
- [ ] 检测能在30秒内稳定
- [ ] 左外侧膀胱经位置正确

### 3. 坐标转换测试
- [ ] 采样10个像素点成功
- [ ] 深度数据有效（至少5个点）
- [ ] 转换为机械臂坐标成功
- [ ] 坐标值在合理范围内（X: 0-600mm, Y: 200-800mm, Z: 100-400mm）

### 4. 机械臂连接测试
```bash
# 设置较大的悬空高度以确保安全
HOVER_HEIGHT_MM=50 ./run_lasttime.sh
```
- [ ] 机械臂连接成功
- [ ] 能移动到安全高度
- [ ] 能移动到第一个点上方

### 5. 单点动作测试
修改代码只执行第一个点的动作：
- [ ] 点筋动作执行成功
- [ ] 分筋动作执行成功
- [ ] 动作幅度符合预期
- [ ] 无碰撞或异常

### 6. 完整序列测试
```bash
./run_lasttime.sh
```
- [ ] 10个点的点筋+分筋全部完成
- [ ] 回到起点成功
- [ ] 顺筋动作完成
- [ ] 返回安全位置成功

### 7. 参数调整测试
```bash
# 测试不同参数组合
HOVER_HEIGHT_MM=30 DIAN_JIN_DEPTH_MM=15 FEN_JIN_LATERAL_MM=25 ./run_lasttime.sh
```
- [ ] 参数正确应用
- [ ] 动作幅度符合设置
